================================================================================
                        CHI TIẾT CÁC THÀNH PHẦN
================================================================================

1. PAGE HEADER
   - Breadcrumb navigation
   - Action buttons (sticky top)
   - Background: White
   - Border-bottom: 1px solid gray
   - Height: 70px

2. IMAGE UPLOAD
   Component:
   - Drag & drop zone
   - Click to upload
   - Preview image
   - Delete button
   - File size validation
   - Image crop/resize option

   Implementation:
   - FileReader API for preview
   - FormData for upload
   - Progress bar during upload
   - Error handling

3. TEXT INPUTS
   Style:
   - Width: 100%
   - Height: 45px
   - Border: 2px solid #E0E0E0
   - Border-radius: 8px
   - Padding: 0 15px
   - Font: 15px
   - Focus: Border blue (#2196F3)
   - Required fields: Red asterisk

4. DROPDOWN/SELECT
   Style:
   - Same as text input
   - Dropdown arrow icon
   - Custom styled dropdown menu
   - Search functionality (for long lists)

5. DYNAMIC LIST (Kỹ năng, Mục tiêu, Tài liệu)
   Features:
   - Add button: Adds new input field
   - Delete button [X]: Removes item
   - Drag handle: Reorder items (optional)
   - Auto-save on blur

   Style:
   - Each item: Card with border
   - Padding: 12px
   - Margin-bottom: 10px
   - Hover: Light background

6. NUMBER INPUTS
   - Input type: number
   - Min: 0
   - Step: 5 or 1
   - Unit display (phút)

7. ACTION BUTTONS

   [LƯU NHÁP]:
   - Background: Gray (#757575)
   - Saves without publishing
   - Success notification

   [XEM THỬ]:
   - Background: Blue (#2196F3)
   - Opens preview in new tab
   - Shows exactly how students will see

   [XUẤT BẢN]:
   - Background: Green (#4CAF50)
   - Validation before publish
   - Confirmation dialog
   - Success notification

   [HỦY]:
   - Background: Red outline
   - Confirmation dialog if changes made
   - Return to dashboard

8. VALIDATION
   Required fields:
   - Họ và tên
   - Trường
   - Môn dạy
   - Lớp
   - Trình độ
   - Ít nhất 1 mục tiêu mỗi loại
   - Thời gian học

   Show error messages:
   - Below field
   - Red color
   - Icon ⚠️

9. AUTO-SAVE
   - Auto-save every 30 seconds
   - Show "Đã lưu tự động" indicator
   - Prevent data loss

10. PREVIEW MODE
    - Opens in new tab
    - Shows exactly client view
    - Can't edit in preview
    - "Quay lại chỉnh sửa" button

================================================================================
                            API ENDPOINTS (GỢI Ý)
================================================================================

POST   /api/admin/lesson-intro/create      - Tạo mới
PUT    /api/admin/lesson-intro/update/:id  - Cập nhật
GET    /api/admin/lesson-intro/:id         - Lấy dữ liệu
POST   /api/admin/upload/image             - Upload ảnh
DELETE /api/admin/lesson-intro/:id         - Xóa
POST   /api/admin/lesson-intro/publish/:id - Xuất bản
