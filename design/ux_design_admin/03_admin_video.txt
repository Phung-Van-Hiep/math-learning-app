                            TECHNICAL IMPLEMENTATION
================================================================================

1. VIDEO STORAGE OPTIONS

   Option A: YouTube
   - Teacher uploads to YouTube first
   - Paste YouTube link
   - Use YouTube embed API

   Option B: Google Drive
   - Teacher uploads to Google Drive
   - Share with public view access
   - Extract file ID
   - Use Drive embed API

   Option C: Direct Upload
   - Upload to server storage
   - Or upload to cloud (AWS S3, Cloudinary)
   - Use video streaming service
   - Generate HLS/DASH streams

2. VIDEO PROCESSING
   - Generate thumbnail automatically
   - Extract metadata (duration, resolution)
   - Create multiple quality versions
   - Add watermark (optional)

3. API ENDPOINTS

   GET    /api/admin/videos              - List all videos
   GET    /api/admin/videos/:id          - Get video details
   POST   /api/admin/videos              - Create new video
   PUT    /api/admin/videos/:id          - Update video
   DELETE /api/admin/videos/:id          - Delete video
   POST   /api/admin/videos/upload       - Upload video file
   POST   /api/admin/videos/validate-url - Validate YouTube/Drive URL
   POST   /api/admin/upload/document     - Upload document
   GET    /api/admin/videos/:id/stats    - Get statistics
