================================================================================
                        CHI TIẾT CÁC THÀNH PHẦN
================================================================================

1. STATISTICS CARDS
   Layout: Grid 4 columns
   Each card:
   - Size: 200x150px
   - Background: Gradient (different colors)
   - Icon: Large, 40px
   - Number: 48px, bold
   - Label: 14px
   - Border-radius: 12px
   - Box-shadow: Medium
   - Hover: Lift effect

2. FILTER & SEARCH
   - Dropdowns: Width 150px, height 40px
   - Search box: Width 300px
   - Border: 2px solid #E0E0E0
   - Focus: Blue border

3. RESULTS TABLE
   Style:
   - Width: 100%
   - Background: White
   - Border: 1px solid #E0E0E0
   - Border-radius: 8px

   Header row:
   - Background: Light gray (#F5F5F5)
   - Font: Bold, 14px
   - Padding: 15px

   Data rows:
   - Padding: 15px
   - Border-bottom: 1px solid #F0F0F0
   - Hover: Light background (#F9F9F9)

   Điểm column:
   - Large, bold
   - Color: Green (≥7), Orange (5-6), Red (<5)
   - Stars: Visual indicator

   Buttons:
   - Small, height 35px
   - Border-radius: 17px
   - Margin: 5px

   Pagination:
   - Bottom of table
   - Numbers clickable
   - Active: Bold, blue

4. PROGRESS CHART
   - Canvas or SVG
   - Line chart với points
   - X-axis: Dates
   - Y-axis: Scores (0-10)
   - Grid lines: Subtle
   - Tooltip on hover
   - Legend: Below chart

5. PERFORMANCE BY TOPIC
   Progress bars:
   - Height: 30px
   - Background: Light gray
   - Fill: Gradient (red to green based on %)
   - Border-radius: 15px
   - Percentage: Right side
   - Topic name: Left side

6. RECOMMENDATIONS
   - Background: Light blue (#E3F2FD)
   - Border-left: 4px solid blue
   - Padding: 25px
   - List với icons
   - Button: Call to action

7. DETAIL MODAL
   - Width: 600px
   - Max-height: 80vh
   - Scrollable content
   - Each question: Expandable
   - Correct: Green checkmark
   - Wrong: Red X + explanation
