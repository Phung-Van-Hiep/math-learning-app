1. PROFILE HEADER
   - Avatar: 120x120px circle
   - Upload button: Opens file picker
   - Name & info: Right side of avatar
   - Background: Light gradient

2. TABS
   - 4 tabs: Info, Password, Notifications, Account
   - Active: Blue background
   - Smooth transition

3. FORM FIELDS
   - Height: 45px
   - Border: 2px solid #E0E0E0
   - Focus: Blue border
   - Disabled fields: Gray background
   - Validation: Real-time

4. PASSWORD STRENGTH
   - Progress bar
   - Colors: Red (weak), Orange (medium), Green (strong)
   - Indicators: Visual + text

5. CHECKBOXES
   - Size: 20x20px
   - Custom styled
   - Label: Clickable
   - Description: Below label, smaller font

6. DANGER ZONE
   - Background: Light red (#FFEBEE)
   - Border: 2px solid red
   - Buttons: Red color
   - Warning icons

7. CONFIRMATION MODAL
   - Password required
   - Clear warning messages
   - Red delete button
   - Cancel button prominent

================================================================================
                        VALIDATION & SECURITY
================================================================================

THÔNG TIN CÁ NHÂN:
- Họ tên: Required, 2-50 chars
- Email: Valid format, unique
- Phone: Valid format (optional)
- Changes require password confirmation (optional)

MẬT KHẨU:
- Current password: Required
- New password: Min 6 chars, strength check
- Confirm: Must match
- Cannot reuse recent passwords

XÓA TÀI KHOẢN:
- Password confirmation required
- Double confirmation
- Email notification
- Grace period: 30 days (optional)
